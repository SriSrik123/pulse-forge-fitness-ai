ExerciseLocation

class ExerciseLocation : Parcelable

Location information during an exercise.

ExerciseType data contains one or more ExerciseSessions. For each ExerciseSession, an exercise route is provided with a set of ExerciseLocation. If the user allows the device's GPS use and exercises outdoors where GPS measurement is possible, then the Samsung Health saves exercise route during the exercise session.

ExerciseLocation consists of the timestamp, longitude, latitude, altitude, and accuracy. It follows the WGS84 coordinate system and includes the following:

PropertyDescription
timestampThe time when the exercise took place.
longitudeThe longitude value of the exercise location.
latitudeThe latitude value of the exercise location.
altitudeThe altitude value of the exercise location.
accuracyThe accuracy of the location.
