ExerciseLog

class ExerciseLog : Parcelable

It specifies the data points measured during the exercise.

The possible fields for the ExerciseLog are time stamp, heart rate, cadence, step count, power and speed. The data present in the ExerciseLog contain measurable values according to the com.samsung.android.sdk.health.data.request.DataType.ExerciseType.EXERCISE_TYPE.
