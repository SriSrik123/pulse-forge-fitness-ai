SkinTemperature

class SkinTemperature : Parcelable

SkinTemperature specifies a single record of skin temperature in skin temperature data. Skin temperature is different than body temperature.

This record includes the following:

  • The skin temperature value as skinTemperature.

  • The lowest skin temperature value as min.

  • The highest skin temperature value as max.

  • The time when measurement started as startTime.

  • The time when measurement ended as endTime.
