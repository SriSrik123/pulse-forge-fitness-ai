HealthDataPoint

class HealthDataPoint : DataPoint

A HealthDataPoint is an individual record of health data saved in the Samsung Health data store.

It has the following basic properties.

PropertyDescription
uidThe data's unique identifier assigned by the Samsung Health.
startTimeThe start time of measurement, specified as Instant.
endTimeThe end time of measurement, specified as Instant.
zoneOffsetThe ZoneOffset for starTime and endTime.
dataSourceThe data's source information including the application package name and which device's ID.
updateTimeThe time when the data was last updated, specified as Instant. It's managed by the Samsung Health.
clientDataIdThe optional data ID supplied by the application using the Samsung Health Data SDK.
clientVersionThe optional data version supplied by the application using the Samsung Health Data SDK.

Other fields vary according to the DataType.
