AuthorizationException

class AuthorizationException(errorCode: Int?, errorMessage: String) : HealthDataException

An exception to indicate that the application is not authorized to perform the specified operation.

This error occurs through application misuse, so it may be corrected in the development stage and ignored at runtime.

The following error codes are defined:

  • ErrorCode.ERR_NO_USER_PERMISSION: This occurs when the data permission was not acquired from the user.

  • ErrorCode.ERR_UNSUPPORTED_OPERATION: This occurs when an unsupported operation is requested.

  • ErrorCode.ERR_ACCESS_CONTROL: This occurs when a calling application is not allowed to use the feature.

  • ErrorCode.ERR_INVALID_PLATFORM_SIGNATURE: This occurs when the Samsung Health application's signature is invalid. This may happen when installing an improper version of the Samsung Health.

  • ErrorCode.ERR_CHILD_ACCOUNT_ACCESS: This occurs when the Samsung account that is set up in the Samsung Health is a child account.
