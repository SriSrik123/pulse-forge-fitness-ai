DualTimeBuilder

class DualTimeBuilder<T : Any> : AggregateRequest.Builder<T>

A dual time builder to create an AggregateRequest instance.

This allows two types of time filters:

  • InstantTimeFilter - with an Instant time interval as a condition. The time interval is represented as a standard time.

  • LocalTimeFilter - with a LocalDateTime time interval as a condition. The time interval is represented as a local date and time.

This class can be instantiated by its builders obtained from the requestBuilder field that each AggregateOperation provides, not directly by a constructor.
