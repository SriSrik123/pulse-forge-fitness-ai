LocalDateBuilder

class LocalDateBuilder<T : Any> : AggregateRequest.Builder<T>

A local date builder to create an AggregateRequest instance.

This allows only LocalDateFilter as a time filter.

This class can be instantiated by its builders obtained from the requestBuilder field that each AggregateOperation provides, not directly by a constructor.
