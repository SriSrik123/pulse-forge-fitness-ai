AssociateBuilder

class AssociateBuilder<T : AssociateTypes>

An associate builder to create an AssociatedReadRequest instance.

This allows filtering the data by using IdFilter.

The instance of this class can be obtained from the associatedReadRequestBuilder field that each data type provides, not directly by a constructor.
