TOTAL_CALORIES_BURNED

@JvmField
val TOTAL_CALORIES_BURNED: AggregateOperation<Float, AggregateRequest.LocalTimeBuilder<Float>>

The aggregate operation to obtain the total value of calories burned in kilocalories.

This helps to create an AggregateRequest instance get the total value of calories burned simply and easily.

This aggregate operation allows the application to set the following:

  • A time filter with LocalTimeFilter.

  • A time filter using LocalTimeFilter along with grouping by LocalTimeGroup.

  • The data's source filter with AggregateSourceFilter.
    AggregateSourceFilter.fromPlatform means obtaining data generated by the Samsung Health.

  • A data order with Ordering.

// set a Local date time range from midnight to now
val startOfDay = LocalDate.now().atStartOfDay()
val now = LocalDateDate.now()

val aggregateRequest = DataType.ActivitySummaryType.TOTAL_CALORIES_BURNED.requestBuilder
    .setLocalTimeFilter(localTimeFilter)
    ... // set some filters or conditions
    .build()
