PredefinedExerciseType

enum PredefinedExerciseType : Enum<DataType.ExerciseType.PredefinedExerciseType>

An exercise type.

It can be used in:

For ExerciseType data's EXERCISE_TYPE

If the ExerciseType data has a single ExerciseSession, the ExerciseType.EXERCISE_TYPE corresponds to the exerciseType of the included exercise session. If there are two or more ExerciseSessions in the ExerciseType data, the ExerciseType.EXERCISE_TYPE indicates a representative exercise type for those ExerciseSessions.

If there is no proper predefined exercise type, set the ExerciseType.EXERCISE_TYPE as OTHER_WORKOUT and set the ExerciseType.CUSTOM_TITLE as a desired exercise name.

For each ExerciseSession's exerciseType

The PredefinedExerciseType value can be set for the ExerciseSession's exerciseType to specify a single exercise activity.

If there is no proper predefined exercise type, set the ExerciseSession.exerciseType as OTHER_WORKOUT and set the ExerciseSession.customTitle as a desired exercise name.
