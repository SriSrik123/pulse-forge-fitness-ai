LAST_BED_TIME

@JvmField
val LAST_BED_TIME: AggregateOperation<LocalTime, AggregateRequest.AllSourceLocalDateBuilder<LocalTime>>

The aggregate operation to obtain the last bed time goal.

This helps to create an AggregateRequest instance get the last bed time goal simply and easily.

This aggregate operation allows the application to set the following:

  • A date filter with LocalDateFilter.

  • A date filter using LocalDateFilter along with grouping by LocalDateGroup.

  • A data order with Ordering.

// set a Local date range with one week ago including today
val today = LocalDate.now()
val localDateFilter = LocalDateFilter.of(today.minusDays(6), today)

val aggregateRequest = DataType.SleepGoalType.LAST_BED_TIME.requestBuilder
    .setLocalDateFilter(localDateFilter)
    .setOrdering(Ordering.DESC)
    // set other conditions
    .build()
