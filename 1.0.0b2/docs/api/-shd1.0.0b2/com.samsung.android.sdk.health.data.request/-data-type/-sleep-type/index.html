SleepType

class SleepType : DataType, DataType.Readable<HealthDataPoint, ReadDataRequest.DualTimeBuilder<HealthDataPoint>> , DataType.ChangeReadable<HealthDataPoint>

The data type representing sleep data specifications.

It includes bedtime, wake-up time, actual sleep duration and sleep score across entire sleep sessions. A single sleep data record contains of one or more sleep session data. Each SleepSession data has the session's start time, end time, sleep duration, and sleep stages.

Sleep data can be measured from wearable devices such as Galaxy Watch, Galaxy Fit, and Galaxy Ring. The Samsung Health's sleep tracker makes it easy to check sleep data based on the sleep goal time set by the user, even if there are multiple sleep sessions.

You can get the following sleep data information:

  • The sleep duration, a bedtime, a wake-up time, and a sleep score

  • The list of sleep sessions

Available aggregate operation

  • TOTAL_DURATION
    : To get a total sleep duration time for a data request.

Data specifications

This data type includes the following information.

PropertyDescription
uid[Mandatory]
The data's unique identifier assigned by the Samsung Health.
startTime[Mandatory]
The Instant time for bed.
endTime[Mandatory]
The Instant time for wake-up.
zoneOffset[Mandatory]
The ZoneOffset for starTime and endTime.
dataSource[Mandatory]
The data's source information including the application package name and which device's ID.
DURATION[Mandatory]
The sleep duration time of this sleep data record.
SESSIONS[Mandatory]
The list of SleepSession.
SLEEP_SCOREThe sleep score of this sleep data record.

Associated data

The Samsung Health records blood oxygen levels and skin temperature data continuously during the user's sleep. The Samsung Health Data SDK enables your application to getting the following associated data.

  • BloodOxygenType data

  • SkinTemperatureType data

For a specific SleepType data record, getting associated data is available through:

  • HealthDataStore.readAssociatedData

  • HealthDataStore.readAssociatedDataAsync
