WaterIntakeGoalType

class WaterIntakeGoalType : DataType

The data type for getting the water intake goal that a user sets in the Samsung Health application.

Predefined instance

Using the following instance allows for obtaining the water intake goal with more concise code.

  • DataTypes.WATER_INTAKE_GOAL

Available aggregate operation

  • LAST
    : To get the last water intake goal.

Required permission

The user's consent is required to access this data type. Check the granted permissions and request permissions if the required permission is not granted yet with:

  • HealthDataStore.getGrantedPermissions

  • HealthDataStore.requestPermissions

See a code example in here. The WaterIntakeGoalType provides read-only access.

Available requests

  • HealthDataStore.aggregateData

  • HealthDataStore.aggregateDataAsync
