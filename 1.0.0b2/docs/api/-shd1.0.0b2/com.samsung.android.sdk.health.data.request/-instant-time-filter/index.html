InstantTimeFilter

class InstantTimeFilter : Parcelable

Filter with an Instant type time interval as a condition. The time interval is represented as a standard time.

This can be used in read, aggregate, and delete requests. However, whether this filter can be used is determined by the request builder provided by each data type.
