LocalDateGroup

class LocalDateGroup : Parcelable

A grouped time interval with a pairing of LocalDateGroupUnit and multiplier.

This means that the grouping is applied to intervals as much as multiplier in local date format.

This can be used in aggregate requests. However, whether this group can be used is determined by the request builder provided by each data type.
