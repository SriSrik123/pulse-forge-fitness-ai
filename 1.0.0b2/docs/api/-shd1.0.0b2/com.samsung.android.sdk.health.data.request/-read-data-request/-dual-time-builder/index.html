DualTimeBuilder

class DualTimeBuilder<T : DataPoint> : ReadDataRequest.Builder<T>

A dual time builder to set the fields and create a ReadDataRequest instance.

This allows for two types of time filters:

  • InstantTimeFilter - with an Instant type time interval as a condition. The time interval is represented as a standard time.

  • LocalTimeFilter - with a LocalDateTime type time interval as a condition. The time interval is represented as a local date and time.

The instance of this class can be obtained from the readDataRequestBuilder field that each data type provides, not directly by a constructor.
