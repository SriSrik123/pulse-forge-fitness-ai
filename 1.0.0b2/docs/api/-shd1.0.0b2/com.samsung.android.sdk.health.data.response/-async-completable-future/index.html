AsyncCompletableFuture

interface AsyncCompletableFuture : Future<Void?>

A Future that accepts a callback which runs code after completion asynchronously as well as blocking await.

The callback has an associated executor or looper, and it is invoked by using this executor or looper once the async query on the future is complete. If the query has already completed when the callback is added, the listener executes immediately.
