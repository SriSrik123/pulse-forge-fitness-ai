abstract fun getDevicesAsync(deviceType: DeviceType): AsyncSingleFuture<List<Device>>

Retrieves a list of Devices that have provided the data stored in the Samsung Health among the devices used, corresponding to specific DeviceType. It works asynchronously using AsyncSingleFuture.
