deleteDataAsync

abstract fun deleteDataAsync(request: DeleteDataRequest): AsyncCompletableFuture

Delete a set of data in the request into the Samsung Health.

In case of deleting one more data, the data deletion process handles each data sequentially. If one data deletion fails, none of them are updated.

Only data provided by the application can be deleted. For example, if the data is recorded by the Samsung Health application, only the Samsung Health application can delete it.

Return

The AsyncCompletableFuture instance telling whether the delete is successful or not.

1.0.0
