readData

abstract suspend fun <T : DataPoint> readData(request: ReadDataRequest<T>): DataResponse<T>

Reads a set of the Samsung Health data according to the specified request.

This method returns the original data point that is not processed or aggregated.

The following code shows how to use the readData function to get a list of heart rate values from a specific period of time:

val readRequest = DataTypes.HEART_RATE.readDataRequestBuilder
    .setLocalTimeFilter(localTimeFilter)
    .build()
val result = healthDataStore.readData(readRequest).dataList
result.forEach { heartRateDataPoint->
    val hr = heartRateDataPoint.getValue(DataType.HeartRateType.HEART_RATE)
    // Process the heart rate value
}
